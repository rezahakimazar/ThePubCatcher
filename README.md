# ThePubCatcher
Catch the publications from Google Scholar
This script catches the publication of researchers based on their Google Scholar ID.
Using the SJR dataset of journals, it not only highlights the publication metrics but also the journal metrics.

V2
In this version of ThePubCatcher, you will be able to see the EEG impact score per paper. Also, the OSF familiarity of the researchers with open science will be fetched from OSF (join date and number of their own projects, registered on OSF)
