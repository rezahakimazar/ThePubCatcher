# ThePubCatcher
Catch the publications from Google Scholar. This script catches the publication of researchers based on their Google Scholar ID. Using the SJR dataset of journals, it not only highlights the publication metrics but also the journal metrics. ThePubCatcher can be used to filter out keywords of interest, making itt a versitile tool for conducting sciencometric research, and observing the development of literature in a field.
